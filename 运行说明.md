# ClearView AI 运行指南

## 前置要求

1. **Node.js** (推荐版本 18 或更高)
   - 下载地址: https://nodejs.org/

2. **Google Gemini API Key**
   - 获取地址: https://ai.google.dev/
   - 需要付费的 Google Cloud Project 才能使用 Veo 视频生成功能

## 运行步骤

### 1. 安装依赖

```bash
npm install
```

### 2. 配置 API Key

编辑 `.env.local` 文件，将 `YOUR_API_KEY_HERE` 替换为你的实际 Gemini API Key：

```
GEMINI_API_KEY=你的实际API密钥
```

### 3. 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:3000` 启动

### 4. 访问应用

在浏览器中打开: http://localhost:3000

## 功能说明

### 图片水印移除模式
- 上传带水印的图片
- 输入自定义提示词（可选）
- 点击 "Remove Watermark" 按钮
- 下载处理后的图片

### 视频恢复模式
- 上传视频的参考帧（截图）
- 第一步：清理参考帧中的水印
- 第二步：输入视频描述
- 第三步：生成并下载恢复后的视频

## 注意事项

1. **API Key 安全**: 不要将 `.env.local` 文件提交到 Git 仓库
2. **费用**: 使用 Google Gemini 和 Veo 会产生 API 调用费用
3. **视频生成**: Veo 视频生成可能需要较长时间（几分钟）
4. **AI Studio 环境**: 如果在 AI Studio 中运行，会自动使用其 API Key 选择功能

## 故障排除

### 端口被占用
如果 3000 端口被占用，Vite 会自动尝试其他端口，查看终端输出即可

### API Key 错误
- 确保 `.env.local` 文件在项目根目录
- 确保 API Key 格式正确（没有多余空格）
- 重启开发服务器

### 依赖安装失败
```bash
# 清除缓存后重新安装
rm -rf node_modules package-lock.json
npm install
```
